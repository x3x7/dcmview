2-D view controls

Key or Mouse inputDescription
Q Quit the progam.

X Zoom out.
R Toggle ROI display, same as ROI button.

Ctrl Mouse_Wheel Change zoom level.
Mouse_Click When on the image display it repositions the cursor (even if not shown). If clicked again in the same spot it centers the image on it (zoom level allowing).

Introduction

You've made it so far as building the program, running it and seeing this page :) Congratulations! Perhaps the visual controls are not self-explanatory but I trust you'll figure them out with a bit of experimentation. There are three other views:

  • 2-D which is also the main control view
  • Histogram the image series histogram with intensity scale controls
  • 3-D with a rudimentary visualization of ROI data, if available

For the 2-D view the controls are show at the top of this page. Since the other views are quite rudimentary, only a brief description is given below.

The Histogram view shows image intensity levels on the Y axis while the gray level values are assigned on the X axis. With the inputs on the vertical area it allows control over which slice of intensities (from minimum to maximum specified) gets linearly mapped into the range of values denoted on the X axis. The intensities below are mapped into a green range of colors, more luminous ones corresponding to lower values and transitioning smoothly into the specified black level. For the intensities above the maximum, the mapping goes into the red color range, with highest intensities being assigned for the darkest shade. The absolute range of intensities are the fixed numbers on the Y axis. To faster change the numeric spinners, in increments of 10, hold the Ctrl key while rotating the mouse wheel.

The 3-D view offers a rudimentary spatial representation of the ROI data. Only keyboard controls are currently available and these are listed in the 2nd table above.
